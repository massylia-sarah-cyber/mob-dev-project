# Mplay

**Mplay - Your Ultimate Music Experience**

A modern Flutter music player application with biometric authentication and beautiful UI.

## 🎵 Features

- 🔐 Biometric Authentication (Fingerprint/Face ID)
- 🎶 Music Streaming & Playback
- ❤️ Favorites Management
- 📊 User Dashboard & Statistics
- 🎨 Modern Glass-morphism Design
- 🌙 Dark Theme Optimized

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (^3.11.0)
- Dart SDK (^3.11.0)
- Firebase account for backend services

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd mplayer
```

2. Install dependencies
```bash
flutter pub get
```

3. Configure Firebase
```bash
flutterfire configure
```

4. Run the app
```bash
flutter run
```

## 🛠️ Tech Stack

- **Framework:** Flutter
- **Language:** Dart
- **State Management:** Provider
- **Backend:** Firebase (Auth, Firestore)
- **Audio:** just_audio, audio_service
- **UI:** google_fonts, flutter_animate
- **Authentication:** local_auth (Biometrics)
